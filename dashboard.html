NETWORK

▼
  • ไม่สามารถเชื่อมต่ออินเทอร์เน็ต

    ไม่สามารถเชื่อมต่ออินเทอร์เน็ต

    สาเหตุที่เป็นไปได้:
    • สาย LAN หลุดหรือเสียบไม่แน่น
    • Router หรือ Modem ปิดหรือเสีย
    • การ์ด LAN เสียหรือไดรเวอร์เสีย
    • ตั้งค่า IP Address ไม่ถูกต้อง
    วิธีแก้ไข:
    • ตรวจสอบสาย LAN ว่าต่อแน่นทั้งสองด้าน
    • ลองรีสตาร์ท Router/Modem (ปิด 30 วินาที แล้วเปิดใหม่)
    • ลองใช้คำสั่ง ipconfig /release และ ipconfig /renew ใน Command Prompt
    • ลองเปลี่ยน DNS Server เป็น 8.8.8.8 และ 8.8.4.4 (Google DNS)
  • WiFi ไม่สามารถเชื่อมต่อได้

    WiFi ไม่สามารถเชื่อมต่อได้

    วิธีแก้ไข:
    • ตรวจสอบว่า WiFi Adapter เปิดอยู่
    • ลองรีสตาร์ท Router
    • ลบ WiFi Network ที่บันทึกไว้แล้วเชื่อมต่อใหม่
    • อัพเดทไดรเวอร์การ์ด WiFi

    คีย์บอร์ดใช้งานไม่ได้

    วิธีแก้ไข:

    • ลองกดปุ่มใดๆ บนคีย์บอร์ดเพื่อออกจากโหมด Sleep
